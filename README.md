# How to Run

Create virtual environment
```
python3 -m venv .env
```

Start your virtual environment
```
. .env/bin/activate (Ubuntu)
.\.env\Script\activate  (Windows)
```

Install requirements
```
pip install -r requirements.txt
```

Run the application
```
python run.py
```

Open on browser
```
http://127.0.0.1:5000/
```
